# Meter readings (schema-driven)

Простое веб-приложение для сбора показаний по схеме из JSON. Интерфейс строится динамически, данные сохраняются в файловую «базу», а пользователь видит только свои записи.

## Быстрый старт
1. Установите зависимости: `npm install express` (остальное использует встроенный Node.js).
2. Запустите сервер: `node server/server.js` (порт по умолчанию 3000).
3. Откройте `http://localhost:3000` в браузере.

## Потоки данных
- Сервер читает структуру формы из `server/schema.json`.
- Отправленные данные пишутся в `server/data/readings.json` с меткой времени и логином пользователя.
- История (`/api/records`) возвращает только записи текущего пользователя (заголовок `x-username`).

## Кастомизация
- Обновите `server/schema.json`, чтобы изменить набор полей. Формат: свойства с типами `string`/`number`, вложенные блоки в массиве `resources`.
- Стили и верстка лежат в `public/style.css` и `public/index.html`.
